# NsFileShare

> 一个用于将Switch中的图片/视频快速保存至手机的App

## 使用方式
1. 在Switch中选择要发送至智能设备的图片/视频文件
2. 手机连接至Switch热点
3. 打开App，点击右上角**刷新**按钮
